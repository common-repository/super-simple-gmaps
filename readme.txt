Super Simple Google Maps Plugin allows you to add Google Maps Widgets to your Site using the Google Maps JS Api

== Description ==

Its a very simple Google Maps Plugin which adds Google Maps support.

== Installation ==

Upload the Simple Google Maps plugin to your blog and activate it. It would work as is.

== FAQ ==

= Is it compatible with latest WordPress? =

Yes, it is, as well as with the latest PHP.
